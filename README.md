# PhoneBook Java
Simple PhoneBook using Java

# Requirements:
Java runtime installed

# Features
- Display Contacts <br>
- Add Contact <br>
- Search Contact <br>
- Remove Contact <br>
- Update Contact 

Contacts are saved into a .txt file

# How to run the application:
Just download the File and open it in your IDE
Run it

Contacts are saved into a .txt file
